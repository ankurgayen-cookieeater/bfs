# **BFS – Buddy For Study**

A minimalist productivity toolkit designed to help students stay organized, focused, and mentally balanced. Built as a lightweight, browser-based app with zero backend and zero setup.

## **Features**
- **Task Splitter** – Break assignments into small, actionable steps.  
- **StudySprint** – A simple 25-minute focus timer with reflections saved locally.  
- **FocusRoom** – Distraction-free mode for deep work sessions.  
- **CarePack** – Quick well-being tips to reset between tasks.

## **Tech Stack**
- HTML, CSS, JavaScript  
- LocalStorage for reflection storage  
- Hosted via GitHub Pages  
- No frameworks, no backend

## **Project Structure**
```
/assets       → logo, visuals  
/components   → tasks, timer, focus, carepack modules  
/utils        → storage + helper utilities  
index.html    
style.css     
script.js
```

## **Running the Project**
Open **index.html** in any modern browser.
